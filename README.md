# 💹 Crypto Tracker

A React app that fetches and displays live cryptocurrency prices with interactive charts.

## 🚀 Features
- Real-time price updates
- Interactive Chart.js graphs
- Responsive design

## 🛠️ Tech Stack
- React (CRA)
- Axios
- Chart.js
- CSS

## 📦 Setup
```bash
git clone https://github.com/ChibuikemLucas/crypto-tracker.git
cd crypto-tracker
npm install
npm start
